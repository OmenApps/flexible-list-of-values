Assumes you have run migrations and created a superuser.

1. Run python manage.py update_lovs to update the default LOV Values in the project to the database.
2. Optionally create another user or two in admin.
3. Create at least one tenant, setting the owner and users fields. The tenant owner can create new LOV Values and choose which Values its Users can select from. If the user account you are logged in with is a tenant owner, then click here to add new LOV Values, and click here to determine which Values your users can choose from.
4. If the user account you are logged in with belongs to a tenant, then click here to choose from among the LOV Value choices the tenant has made available to you.
